Positions in CSS

  1. Static: the original or default position of an element.
  2. Relative: Element is positioned relative to it's original/normal position. Same as static positioning, just that here we can move the element by adding offset values. Element stays in the normal document flow.
  3. Absolute: Element is positioned relative to it's parent. If parent is not present then element is positioned relative to the main document. Element is removed from the normal document flow, no space is created for the element.
  4. Fixed: Element is always positioned relative to the viewport. Element always stays in the same position even on scrolling. Element is removed from normal document flow.
  5. Sticky: Element is positioned according to the normal document flow. Till the elemnt hits the top, it has normal position, after it touches the top it's position becomes fixed.
